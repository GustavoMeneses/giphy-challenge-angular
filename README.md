# Giphy Challenge

Application built with Ionic + Angular + NestJS + PostgreSQL

## Requirements

- Node >= 16
- NestJS - 9
- Ionic - 6
- PostgreSQL >= 15
- NPM
- Yarn

## Setting up the environment

Follow these steps to install Ionic: [Installing Ionic](https://ionicframework.com/docs/intro/cli#install-the-ionic-cli)
